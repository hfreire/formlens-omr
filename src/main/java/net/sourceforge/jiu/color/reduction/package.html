Classes to convert images to a lower color type. Usually, that conversion is lossy (e.g. from RGB truecolor to grayscale, or from shades of gray to black and white).

However, the {@link net.sourceforge.jiu.color.reduction.AutoDetectColorType} class reduces only if an image can be converted to a lower color type without losing information. This is the inverse operation to color promotion, provided by another package.
